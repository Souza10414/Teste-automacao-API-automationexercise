# Teste-automa-o-API-automationexercise
Projeto de automação de testes de interface e API da aplicação AutomationExercise, desenvolvido com Cypress. Inclui testes E2E e validação de APIs utilizando os métodos GET, POST, PUT e DELETE. Utiliza Faker para dados dinâmicos e Mochawesome para geração de relatórios, seguindo boas práticas de QA.
